# Utils

`format.js` — formatlash funksiyalari.

## Funksiyalar
- `formatDate(date)` → `DD.MM.YYYY` yoki `-`
- `formatMoney(amount, currency)` → `1,234 USD`
- `formatM3(val)` → `0.1234 m³`
- `calcM3PerPiece(thickness, width, length)` → m³ (hamma mm da)
- `statusColors` — kelyapti/faol/sotildi → Ant Design tag ranglari
- `statusLabels` — kelyapti/faol/sotildi → O'zbekcha nomlar
